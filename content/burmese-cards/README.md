## 🎯 How to Play

1. **Pack ရွေးပါ** - Friends, Couple, Family, Classic ထဲမှ တစ်ခု ရွေးပါ
2. **Difficulty ရွေးပါ** - Easy, Medium, Hard ထဲမှ တစ်ခု ရွေးပါ
3. **Player တစ်ယောက်ကို** Truth (သို့မဟုတ်) Dare ရွေးခိုင်းပါ
4. **Truth ဖြေရင်** အမှန်ဖြေရပါမယ်
5. **Dare လုပ်ရင်** အဆုံးထိလုပ်ရပါမယ်
6. **မလုပ်ချင်ရင်** Challenge ပြန်ပေးလို့ရပါတယ်
